The presentation is created using **Sli.dev**.

### **Steps to Run the Presentation**

1.  **Navigate to the presentation folder:**
    ```bash
    cd presentation
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the presentation:**
    ```bash
    npm run dev
    ```

This will open the presentation in your browser at `http://localhost:3030`.
